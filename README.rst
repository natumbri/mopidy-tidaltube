Mopidy extension for playing Tidal playlists using mopidy-youtube extension


Installation
============

Install by running::

    python3 -m pip install Mopidy-TidalTube

See https://mopidy.com/ext/tidaltube/ for alternative installation methods.


Configuration
=============
